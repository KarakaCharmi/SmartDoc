# How to Use Summarize Feature - Step by Step Guide

## Overview
The Summarize feature lets you quickly create condensed versions of any text from your documents. It's perfect for getting key points without reading everything.

---

## 📋 Step-by-Step Instructions

### **Step 1: Upload a Document**
1. Click **"Upload Document"** or the **upload button** in the app
2. Select a PDF, Word (DOCX), or Text file from your computer
3. Wait for the upload to complete (you'll see a success message)
4. Document appears in your "My Documents" list

---

### **Step 2: Open the Document**
1. Click on the document name from your list
2. The document will open (you'll see a preview or viewer)
3. The document is now ready to work with

---

### **Step 3: Select Text to Summarize**
Choose one of these two methods:

#### **Method A: Copy & Paste Text (Easiest)**
1. Find the text you want to summarize in the document
2. Highlight/select the text
3. Copy it (Ctrl+C or Cmd+C)
4. Go to the **"Summarize"** section of the app
5. Paste the text into the **"Select text or paste here"** box
6. Click **"Summarize"** button

#### **Method B: Manual Text Entry**
1. Read through your document
2. Identify the section you want summarized
3. Type or paste it directly into the text box in the Summarize section

---

### **Step 4: Choose Summary Style (Optional)**
Select how detailed you want the summary:
- **"Concise"** (default) - Short bullets, 120-180 words ⭐ Recommended
- **"Detailed"** - More thorough, 200-300 words
- **"Short"** - Very brief, 80-120 words

Toggle **"Use bullets"** ON/OFF:
- **ON** (default) - Summary as bullet points ✓
- **OFF** - Summary as paragraphs

---

### **Step 5: Generate Summary**
1. Click the **"Generate Summary"** or **"Summarize"** button
2. Wait 5-15 seconds (system is thinking...)
3. Your summary appears in the results area

---

### **Step 6: Use the Summary**
You can now:
- ✓ Read the key points
- ✓ Copy the summary to use elsewhere
- ✓ Save it to your notes
- ✓ Share the summary with others
- ✓ Generate another summary with different settings

---

## 💡 Examples: What to Put in the Text Box

### **Example 1: Product Manual**
**Original text** (1000+ words):
```
The XYZ-3000 printer is a multifunction device that combines printing, 
scanning, and copying capabilities. To set it up, first ensure you have 
all cables connected properly. The power cable should be plugged into a 
grounded outlet. Next, install the toner cartridge by opening the front 
panel and sliding the cartridge into the designated slot until you hear 
a click...
```

**What to ask**: Copy the entire setup section (2-5 paragraphs)

**Resulting summary**:
```
• XYZ-3000 is a multifunction printer (print, scan, copy)
• Connect all cables to grounded outlet
• Install toner by sliding cartridge into slot until clicking
• Download drivers from manufacturer website
• Run installation wizard to complete setup
```

---

### **Example 2: Research Article**
**Original text** (long technical section):
```
Photosynthesis is a complex biochemical process that occurs in plant 
chloroplasts, involving two main stages: light-dependent reactions and 
the Calvin cycle. During light reactions, chlorophyll absorbs photons, 
creating electrons that drive ATP and NADPH synthesis. In the Calvin cycle, 
these products drive CO2 fixation and glucose synthesis...
```

**What to ask**: Copy the mechanism section (1-3 paragraphs)

**Resulting summary**:
```
• Photosynthesis has two stages: light reactions + Calvin cycle
• Light reactions: chlorophyll absorbs photons → ATP + NADPH produced
• Calvin cycle: uses ATP/NADPH to fix CO2 into glucose
• Occurs in plant chloroplasts
```

## ❓ FAQ

### **Q: How long can the text be?**
A: You can summarize up to ~25,000 words at once. For very long documents:
- Summarize section by section
- Copy 2-5 paragraphs at a time
- This keeps summaries focused and accurate

### **Q: What if the summary is wrong or incomplete?**
A: Try these fixes:
- Paste a different section and try again
- Use "Detailed" style instead of "Concise"
- Make sure the text is clear and grammatically correct
- Try "Short" style if it's too wordy

### **Q: Can I summarize in a different language?**
A: Yes! Paste text in any language (English, Spanish, French, Chinese, etc.)
- The summary will be in the same language as your source text

### **Q: How long does it take?**
A: Typically 5-15 seconds depending on:
- Text length (longer = slightly slower)
- Server load
- Your internet speed

### **Q: Can I save the summary?**
A: Currently summaries display on screen. To save:
- Copy the summary → paste into a Note app, Word doc, or email
- Take a screenshot
- Feature coming soon: Save summaries directly in SmartDocQ

## 🎯 Pro Tips

### **Tip 1: Get Better Summaries**
- **DO**: Paste complete paragraphs (not random sentences)
- **DON'T**: Paste tables, images, or formatting-heavy text
- **DO**: Include context (background info at the start helps)

### **Tip 2: Use the Right Style**
- **For reports**: Use "Detailed" → get all key points
- **For quick overview**: Use "Concise" → bullet points
- **For very busy**: Use "Short" → essential facts only

### **Tip 3: Multiple Summaries**
You can:
1. Generate a "Concise" summary in bullets
2. Then generate a "Detailed" summary in paragraphs
3. Use both versions—one for reading, one for sharing

### **Tip 4: Combine with Quiz/Flashcards**
1. Generate a summary first
2. Use that summary to create quiz questions
3. Use those questions to make flashcards
4. Study from flashcards for better learning

## 📱 Using the "Ask" Feature (Different from Summarize)

### When to Use "Ask" Instead of "Summarize":
- **Summarize**: "Give me key points of this section"
- **Ask**: "How do I install the printer?" or "What are the side effects?"

### How to Use Ask:
1. Upload a document
2. Go to **"Chat/Ask"** section
3. Type a question like:
   - "What is the main topic?"
   - "How does this work?"
   - "What are the 3 most important points?"
   - "Explain this like I'm 10 years old"
4. Click **"Send"** or **"Ask"**
5. Get an answer based on the document

### Ask vs Summarize Comparison:

| Feature | Summarize | Ask |
|---------|-----------|-----|
| **Purpose** | Get overview | Answer specific question |
| **When to use** | Don't know what to expect | Know what you want |
| **Example input** | Raw text | "What is X?" |
| **Result** | Key points, bullets | Direct answer |
| **Speed** | ~10 seconds | ~5 seconds |
