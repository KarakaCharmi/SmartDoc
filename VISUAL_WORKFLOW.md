## 📋 Summary Generation Examples

### Example 1: Long Article
```
INPUT (original - 1200 words):
────────────────────────────────────────────────────────
"Artificial intelligence has been rapidly advancing over the past 
decade. Machine learning, a subset of AI that allows computers to 
learn from data, has become increasingly sophisticated. There are 
several types of machine learning including supervised learning, 
unsupervised learning, and reinforcement learning. In supervised 
learning, the algorithm learns from labeled data..."
[continues for 1200 more words...]

PROCESS:
────────────────────────────────────────────────────────
1. Copy full article (1200 words)
2. Paste into Summarize box
3. Select "Concise" + "Bullets"
4. Click Generate

OUTPUT (summary - 150 words):
────────────────────────────────────────────────────────
• Artificial intelligence rapidly advancing in past decade
• Machine learning: subset of AI that learns from data
• Three types: supervised, unsupervised, reinforcement
• Supervised learning: algorithm learns from labeled data
• Unsupervised learning: finds patterns in unlabeled data
• Reinforcement learning: learns through rewards/penalties
• Applications: image recognition, natural language processing
• Future impact: automation, new job roles, efficiency gains
```

## 🚨 Common Issues & Solutions

```
Issue                          Solution
────────────────────────────────────────────────────────
Summary generation fails       → Try shorter text section
Summary seems inaccurate       → Use "Detailed" style
Takes too long (>30s)          → Server busy, try again later
Can't upload file              → File too large or format unsupported
Button doesn't respond         → Refresh page & retry
Summary is too short           → Use "Detailed" instead of "Short"
Summary is too long            → Use "Short" instead of "Detailed"
```
