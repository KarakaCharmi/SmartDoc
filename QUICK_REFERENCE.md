# Quick Reference: Summarize Feature

## ⚡ 5-Minute Quick Start

### Upload Document
```
Upload → Select file (PDF/Word/TXT) → Wait for success
```

### Generate Summary
```
1. Open document from your list
2. Copy paragraph(s) from the document  
3. Click "Summarize" button
4. Paste text in the box
5. Choose: Concise | Detailed | Short
6. Toggle: Bullets ON/OFF
7. Click "Generate Summary"
8. Get results in 10-15 seconds
```

---

## 📝 What to Paste in the "Summarize" Box

### ✅ GOOD Examples:
- "Here is the introduction to my research paper..."
- "The manufacturing process works as follows: [full paragraph]"
- "Meeting notes from today's standup..."
- "This is the abstract from the article..."

### ❌ AVOID:
- Random disconnected sentences
- Just a title or heading
- Corrupted/unreadable text
- Tables or raw data (without context)

---

## 🎯 Ask vs Summarize

| What You Want | Use This | Example |
|---------------|----------|---------|
| Overview/key points | **SUMMARIZE** | "Give me the main ideas" |
| Specific answer | **ASK** | "How do I fix the error?" |
| Short bullets | **SUMMARIZE** | "Quick summary" |
| In-depth answer | **ASK** | "Explain step by step" |

---

## 💬 Example Questions for "Ask" Feature

### Good Questions to Ask:
```
"What is the main topic?"
"How does photosynthesis work?"
"What are the side effects?"
"Summarize this in 3 sentences"
"What is the biggest problem mentioned?"
"List all the steps"
"What should I do first?"
"Explain this more simply"
```

### What to Type in Ask Box:
✓ "How do I..."  
✓ "What is..."  
✓ "Explain..."  
✓ "List the..."  
✓ "Why..."  
✓ "When..."

## ⚙️ Settings

**Style Options:**
- **Concise** ⭐ (5-8 bullets, ~140 words) - Best for quick overview
- **Detailed** (8-12 bullets, ~250 words) - All important points  
- **Short** (3-5 bullets, ~100 words) - Essential only

**Format:**
- **Bullets ON** ✓ (easier to scan)
- **Bullets OFF** ✗ (paragraph format)

## 🔄 Workflow Examples

### 📚 Student Study:
```
1. Upload textbook chapter (PDF)
2. Summarize → "Concise" style
3. Get key points as bullets
4. Generate quiz from those points
5. Create flashcards
6. Study flashcards
```

### 📊 Business Report:
```
1. Paste report section
2. Summarize → "Detailed" style  
3. Share summary in email
4. Ask specific questions as needed
```

### 📖 Reading Article:
```
1. Copy article text
2. Summarize → "Short" style
3. Get 2-minute overview
4. Decide if you want full article
```

## Typical Results

You paste this (500 words):
```
Artificial intelligence is transforming how we work and live. Machine learning, a subset of AI, enables computers to learn from data without being explicitly programmed. Deep learning uses neural networks with many layers...
```

You get this (2-minute read):
```
• AI is changing work and lifestyle
• Machine learning: computers learn from data automatically
• Deep learning uses multi-layer neural networks
• Applications: image recognition, natural language processing
• Impact: increased automation, new job categories emerging
```

## ✅ Troubleshooting

| Problem | Solution |
|---------|----------|
| "No summary generated" | Text may be too short or unclear - try longer section |
| "Summary seems wrong" | Try different style or rephrase your text |
| "Takes too long" | Server might be busy - try again in 30 seconds |
| "Can't upload file" | File might be too large (max 25 MB) or unsupported format |

---

## 🎓 Best Practices

1. **Paste 2-5 paragraphs** for best results
2. **Use "Concise"** by default, "Detailed" if you need more
3. **Try multiple times** with different sections
4. **Use Ask** when you have a specific question
5. **Combine with Quiz** to test your understanding
